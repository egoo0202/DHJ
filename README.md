# 一個迴路（12 芯）示意圖
                      
    （12 芯光纖主纜）
    ┌───────────────────┐
    │    12 Core Fiber    │ F1-F12分別代表一條光纖主纜中的12芯，A/B代表通一芯線兩端點
    └───────────────────┘
   
機櫃 A 端 ──F1A──┐
                 │   主用纖芯（TX/RX）
機櫃 B 端 ──F1B──┘
                 │───── 一條完整光路（TX→RX）

機櫃 A 端 ──F2A──┐
                 │   主用纖芯（第二股，雙工配對）
機櫃 B 端 ──F2B──┘
                 │───── 一條完整光路（TX→RX）


機櫃 A 端 ──F3A──┐
                 │   備援纖芯（TX/RX）
機櫃 B 端 ──F3B──┘
                 │───── 備援光路


機櫃 A 端 ──F4A──┐
                 │   備援纖芯（第二股）
機櫃 B 端 ──F4B──┘
                 │───── 備援光路


F5  ─── 未熔接（Dark Fiber）
F6  ─── 未熔接（Dark Fiber）
F7  ─── 未熔接（Dark Fiber）
F8  ─── 未熔接（Dark Fiber）
F9  ─── 未熔接（Dark Fiber）
F10 ─── 未熔接（Dark Fiber）
F11 ─── 未熔接（Dark Fiber）
F12 ─── 未熔接（Dark Fiber）


（中間全部串過所有 Skid）
 A端 → Skid1 → Skid2 → … → SkidN → B端

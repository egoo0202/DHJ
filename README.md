# 東海金FLN機櫃光纖盒佈局規劃
                      
    ┌───────────────────────────────────────────────────────────────────┐
    │    每個迴路各有一條12芯光纖串接，光纖的頭(A端)跟尾(B端)都連接回機櫃    │                                                                                                        │    LC為光纖盒接頭簡稱，L1-L4代表4個電池區迴路                        |
    │    F1-F12代表一條光纖中的12根芯線                                   |
    └───────────────────────────────────────────────────────────────────┘

# 安南FLN機櫃光纖盒實例 
# 一個光纖盒有12排，每排上下兩組LC Duplex
# 左6排保留給A端 (連接Switch02)，右6排保留給B端 (連接Switch01)

東海金光纖盒佈局規劃 （仿照安南）

12 排 × 每排上下2組LC Duplex

左6排保留給A端 (連接Switch02)，右6排保留給B端 (連接Switch01)

差異在於每個迴路多熔接一組備用芯線

────────────────────────────────────────────

光纖盒左側熔接芯線A端

直排 01  【Loop 1 – A 端】

  上：LC01  主用  L1-F1A / L1-F2A
  
  下：LC02  備援  L1-F3A / L1-F4A

直排 02  【預留緩衝（空）】

  上：LC03  空
  
  下：LC04  空

直排 03  【Loop 2 – A 端】

  上：LC05  主用  L2-F1A / L2-F2A
  
  下：LC06  備援  L2-F3A / L2-F4A

直排 04  【Loop 3 – A 端】

  上：LC07  主用  L3-F1A / L3-F2A
  
  下：LC08 備援  L3-F3A / L3-F4A

直排 05  【預留緩衝（空）】

  上：LC09  空
  
  下：LC10  空

直排 06  【Loop 4 – A 端】

  上：LC11  主用  L4-F1A / L4-F2A
  
  下：LC12  備援  L4-F3A / L4-F4A

─────────────  ← 第 6、7 排之間本來就有間隔  ─────────────

光纖盒右側熔接芯線B端

直排 07  【Loop 1 – B 端】

  上：LC13  主用  L1-F1B / L1-F2B
  
  下：LC14  備援  L1-F3B / L1-F4B

直排 08  【預留緩衝（空）】

  上：LC15  空
  
  下：LC16  空

直排 09  【Loop 2 – B 端】

  上：LC17 主用   L2-F1B / L2-F2B
  
  下：LC18 備援   L2-F3B / L2-F4B

直排 10 【Loop 3 – B 端】

  上：LC19  主用  L3-F1B / L3-F2B
  
  下：LC20  備援  L3-F3B / L3-F4B


直排 11 【預留緩衝（空）】

  上：LC21  空
  
  下：LC22  空
  
直排 12 【Loop 4 – B 端】
  
  上：LC23  主用  L4-F1B / L4-F2B
  
  下：LC24  備援  L4-F3B / L4-F4B

────────────────────────────────────────────

已使用：Loop1~4 共 16 個 LC（32 芯接點）

預留：  8 個 LC（16 芯接點）分散在中間緩衝區（或是後面也行）

依據FLN通訊圖，A端會跳線連接到Switch02，B端會跳線連接到Switch01
